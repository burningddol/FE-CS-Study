# UI kit · 강의 목록 페이지 (Course list)

강의 전체 목록을 탐색할 수 있는 메인 허브 페이지. 원본 위키의 6단계 커리큘럼을 컬러 태그·필터·카드 그리드로 풀어냈어요.

## 파일

- `index.html` — 프로토타입 진입점. 필터 클릭, 검색, 정렬 토글 동작.
- `Nav.jsx` — 상단 sticky 내비.
- `HeroBanner.jsx` — 상단 히어로 + 진행률 요약.
- `StageFilter.jsx` — 단계별 컬러 필터 칩.
- `CourseCard.jsx` — 강의 카드.
- `FilterBar.jsx` — 정렬 · 검색 바.
- `Footer.jsx` — 하단 푸터.

## 포함하지 않은 것

- 로그인/회원가입, 결제, 리뷰 페이지는 UI 킷 범위 밖입니다 (사용자 요청: 강의 목록 + 강의 상세 2종).

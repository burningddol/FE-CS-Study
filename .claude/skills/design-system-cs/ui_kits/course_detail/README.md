# UI kit · 강의 상세 / 학습 페이지 (Course detail)

한 챕터를 실제로 학습할 때 보게 되는 페이지. 영상 플레이어, 챕터 개요, 좌측 커리큘럼 목차, 진행률, 다음 강의 CTA.

## 파일
- `index.html` — 프로토타입 진입점
- `TopBar.jsx` · `Breadcrumb.jsx`
- `LessonPlayer.jsx` — 영상 플레이어 mock
- `LessonMeta.jsx` — 제목/메타/액션
- `LessonBody.jsx` — 본문(요약/키워드/탭)
- `CurriculumSidebar.jsx` — 좌측 커리큘럼 목차
- `NextLessonCTA.jsx`
